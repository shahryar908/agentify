üîó API Endpoints

GET / - Root Endpoint

Get API information and available endpoints

curl -X GET "http://localhost:8000/"

GET /health - Health Check

Check API health status and agent count

curl -X GET "http://localhost:8000/health"

POST /demo/create-sample-agent - Create Demo Agent

Create a sample math agent for testing

curl -X POST "http://localhost:8000/demo/create-sample-agent"

POST /agents - Create Agent

Create a new AI agent with custom configuration

curl -X POST "http://localhost:8000/agents" \ -H "Content-Type: application/json" \ -d '{ "name": "My Custom Agent", "description": "A custom agent for specific tasks", "api_key": "your-groq-api-key" }'

GET /agents - List All Agents

Get a list of all created agents

curl -X GET "http://localhost:8000/agents"

GET /agents/{agent_id} - Get Agent Info

Get detailed information about a specific agent

curl -X GET "http://localhost:8000/agents/{agent_id}"

POST /agents/{agent_id}/chat - Chat with Agent

Send a message to an agent and get a response

curl -X POST "http://localhost:8000/agents/{agent_id}/chat" \ -H "Content-Type: application/json" \ -d '{ "message": "What is 25 + 17?" }'
Example Response:
{"response": "25 + 17 = 42", "agent_id": "...", "tools_used": true}

GET /agents/{agent_id}/tools - Get Agent Tools

List all available tools for a specific agent

curl -X GET "http://localhost:8000/agents/{agent_id}/tools"

DELETE /agents/{agent_id} - Delete Agent

Delete a specific agent

curl -X DELETE "http://localhost:8000/agents/{agent_id}"

üßÆ Available Tools

The math agent comes with the following built-in tools:

  • add_numbers - Add two numbers together
  • multiply_numbers - Multiply two numbers
  • divide_numbers - Divide first number by second
  • subtract_numbers - Subtract second number from first
  • calculate_power - Calculate base raised to power
  • calculate_square_root - Calculate square root
